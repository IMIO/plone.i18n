Description

    Plone i18n is a package that provides various advanced features for
    internationalization and localization as they are usually needed in
    Content Management Systems like Plone.

Requirements

  * Python >= 2.4

  * Zope >= 2.10.x

Installation

    For now you have to manually place the i18n folder in the lib/python/plone
    directory of your Zope instance and restart the server.
